Upsell Tracker

Theo Carnavos
April 4th, 2016

Project Goals:

  • Create an environment for managing upsells, improving over spreadsheet only
    • Older spreadsheet method relied on Excel formulas and manual checks
    • Users were entering placeholder values which overstayed their welcome
    • Large amounts of time were spent collecting and processing individual user data, with no large scale view
  • Develop understanding of the AngularJS framework and Google Materialize
  • Practice with user experience design

Demo

Time for a live demo!

Lessons Learned:

  • Many features take longer than expected and/or must be revisited throughout the project
  • Brainstorming use cases in planning helps limit the amount of backtracking.
